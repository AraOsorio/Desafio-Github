Diseño de experiencia
para todos

Creo identidades visuales y experiencias digitales que reflejan la esencia de cada marca. Mi enfoque combina estrategia, empatía y creatividad para transformar ideas en soluciones visuales que comunican con claridad y generan una conexión auténtica con el público.

Acerca de mi

Diseño Gráfico

Creo piezas visuales que comunican con propósito. Desde identidades de marca hasta contenido digital, fusiono estética, mensaje y emoción para lograr diseños coherentes y memorables.

Diseño UX/UI

Diseño experiencias digitales centradas en las personas. Analizo, estructuro y creo interfaces intuitivas que combinan funcionalidad, usabilidad y belleza visual para generar una conexión real con el usuario.

Diseño Web

Desarrollo sitios web modernos, adaptativos y visualmente atractivos. Cada proyecto está pensado para transmitir identidad, optimizar la experiencia del usuario y potenciar la presencia digital de las marcas.

Mis trabajos

Proyecto 1

Rediseño de identidad visual. Se trabajó en la creación de una nueva línea gráfica, paleta de color y piezas impresas que reflejan una imagen más moderna y coherente con su público lector.

Proyecto 2

Diseño UX/UI para una aplicación móvil enfocada en la meditación y el autocuidado. El proceso incluyó investigación de usuarios, creación de wireframes y desarrollo de una interfaz limpia, accesible y relajante.

Proyecto 3

Diseño web responsivo para un estudio de arte contemporáneo. Se priorizó la experiencia visual, el uso del color y la navegación fluida para destacar el portafolio artístico y mejorar la interacción del usuario.
